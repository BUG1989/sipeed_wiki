teedoc official website
=====

visit: [teedoc.github.io](https://teedoc.github.io)


## build locally

* Install teedoc

```
pip3 install teedoc
```

* Get site source files

```
git clone https://github.com/teedoc/teedoc_site
```

* Install plugins

```
cd teedoc_site
teedoc install
```

* build and serve

```
teedoc build
teedoc serve
```

then visit [http://127.0.0.1:2333](http://127.0.0.1:2333)
