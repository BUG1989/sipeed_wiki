# Tang Nano 系列例程

- 编辑于2022.04.18

这里汇总Tang nano例程，持续更新

## 前言

对于Programmer(下载程序)建议使用 
https://dl.sipeed.com/shareURL/TANG/programmer 里面的文件。
下载后解压替换掉Gowin对应安装目录的Programmer文件夹即可。
不会替换的话可以在下载解压后的Programmer程序中手动添加对应下载文件来进行烧录。

另外欢迎投稿或者提供历程，有什么例程想法的话也可以提出。
